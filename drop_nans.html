How-To Guide: Handling Missing Data

1.1 Handling Missing Data

  • i. Function to drop rows/columns with missing values

    Here is an example function to drop rows or columns with missing values. You can find the code here.

Sample Data

ID Name Age Email
1 John Doe 28 john@example.com
2 Jane Smith jane@example.com
3 Emily Jones 34

Code Snippet

def drop_missing_values(df, axis=0): """ Drops rows or columns with missing values. :param df: DataFrame :param axis: 0 for rows, 1 for columns :return: DataFrame with missing values dropped """ return df.dropna(axis=axis)
